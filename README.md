# Api-Labs

## Descripción

Api-Labs es una API diseñada para agilizar la gestión de quejas y su resolución a través de un rol de pasante. Está específicamente dirigido a laboratorios universitarios.

## Características

- Gestión de Quejas: Permite a los usuarios enviar y seguir el estado de sus quejas.
- Acceso Basado en Roles: Proporciona diferentes niveles de acceso y control basados en roles de usuario (por ejemplo, pasantes, gerentes de laboratorio).
- Seguimiento de Resoluciones: Mantiene un registro de cómo se resuelven las quejas, proporcionando transparencia y responsabilidad.

## Instalación

Clona el repositorio e instala las dependencias:

```bash

git clone https://github.com/tu-repo/Api-Labs.git
cd Api-Labs
npm install

```

## Uso

Inicia el servidor:

```bash

npm start

```
